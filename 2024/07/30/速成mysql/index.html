二、外键的添加方式
　　1、方式一
　　　　create table 表名 (
　　　　　　字段名1 数据类型  约束，
　　　　　　字段名2 数据类型 约束，
　　　　　　。。。
　　　　　　constraint  外键名　　foreign key（字段名）　　references 　　主表名（字段名）　　级联操作）；
　　　方式2:表创建完成后添加外键
　　　alter table　表名 add constraint  外键名  foreign key (字段1） references  表名2（字段名2）级联操作；

在数据表中创建外键使用 FOREIGN KEY 关键字，具体的语法规则如下：
[CONSTRAINT <外键名>] FOREIGN KEY 字段名 [，字段名2，…]
REFERENCES <主表名> 主键列1 [，主键列2，…]
其中：外键名为定义的外键约束的名称，一个表中不能有相同名称的外键；字段名表示子表需要添加外健约束的字段列；主表名即被子表外键所依赖的表的名称；主键列表示主表中定义的主键列或者列组合。
